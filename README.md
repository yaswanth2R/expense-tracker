# Expense Tracker

## Backend

### Install

```bash
cd backend
npm install
```

### Run

```bash
npm start
```

### API Endpoints

- `POST /api/auth/login`
- `POST /api/auth/register`
- `GET /api/categories`
- `POST /api/categories`
- `DELETE /api/categories/:name`
- `GET /api/expenses`
- `POST /api/expenses`
- `PUT /api/expenses/:id`
- `DELETE /api/expenses/:id`
- `GET /api/expenses/monthly?month=YYYY-MM`
- `GET /api/expenses/trends`
